According to the "Phish Companion", there is some uncertainty about the set breaks.
Trey definitely announces a set break after "Alumni Blues", so that we know for sure.
However, it seems as though the taper paused the original tape deck when there were long breaks between songs.
Because of that, it remains unclear exactly where the break was between sets 1 and 2.
I have "You Enjoy Myself" starting set 2, as that is consistent with the "Phish Companion".
